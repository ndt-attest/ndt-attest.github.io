Аттестация специалистов

по неразрушающему контролю с последующей сертификацией через сертификационный центр

персонала по НК при Болгарском обществе дефектоскопии (БОД)EN 4179:2017

О центре аттестации

ООО «НПФ «Диагностические приборы» занимается организацией, проведением и оформлением результатов квалификационных экзаменов специалистов по неразрушающим методам контроля на 1, 2 и 3 уровни квалификации с целью дальнейшей сертификации через Сертификационный центр персонала по неразрушающему контролю (СЦП ПНК) при болгарском обществе по дефектоскопии (БОД) (г. София, Республика Болгария) в соответствии с требованиями EN 4179: 2017 «Аэрокосмическая серия. Квалификация и одобрение персонала неразрушающего контроля».

 Компетентность АЦНК подтверждено сертификатом, выданным сертификационным центром персонала по неразрушающему контролю при Болгарском обществе по дефектоскопии. Компетентность СЦП ПНК подтверждено сертификатом аккредитации согласно EN ISO 17024 для сертификационной схемы по EN ISO 9712, выданным Болгарским сервисом аккредитации № 1 ОСП 30.12.2014г., Сертификатом № 3 от 20.06.2014р. Болгарского национального авиационно-космического Борда по персоналу НК для сертификационной схемы согласно EN 4179. (http://www.ndt-bg-cert.org/).

СЦП ПНК имеет сертификат Европейской федерации неразрушающего контроля № 7003 о регистрации участия в соглашении о взаимном признании сертификации (выданные им сертификаты признаются во всех государствах-членах Европейского Союза и в следующих странах: Австралии, Беларуси, Хорватии, Израиле, Норвегии, России, Швейцарии и Украине). 

 Квалификационные экзамены специалистов по НК проводят представители (экзаменаторы) Сертификационного центра персонала по неразрушающему контролю при Болгарском обществе дефектоскопии на базе АЦНК ООО «НПФ« Диагностические приборы ». Экзаменаторами СЦП ПНК есть высококвалифицированные специалисты НК, сертифицированные на 3-й уровни квалификации в соответствии с EN 4179, которые имеют большой опыт проведения квалификационных экзаменов, а также практической работы в проведении НК продукции аэрокосмической отрасли. По желанию Заказчика, квалификационные экзамены могут проводиться и на технической базе Заказчика, с соблюдением всей процедуры проведения квалификационных экзаменов, по предварительному согласованию всех нюансов с СЦП ПНК. 

 АЦНК укомплектован современными приборами и оборудованием ведущих отечественных и зарубежных производителей для проведения квалификационных экзаменов специалистов по НК. АЦНК обладает необходимым фондом европейской и международной нормативной и методической документации по НК, позволяет обеспечить процесс проведения квалификационных экзаменов. Экзаменационные образцы, которые используются для проведения квалификационных экзаменов, представляют собой реальные фрагменты изделий или деталей и узлов авиационной техники. 

 АЦНК осознает важность беспристрастности при выполнении процедур аттестации, решает конфликты интересов и обеспечивает объективность своей аттестации. Сертификация может пройти любой кандидат, если он соответствует требованиям по состоянию здоровья, специальной подготовки и стажа практической работы в соответствии с требованиями стандарта EN 4179 из следующих методов НК:

  • капиллярный контроль (PT)
  • магнитопорошковый контроль (MT)
  • вихретоковый контроль (ET)
  • ультразвуковой контроль (UT)

Процедура аттестации АЦНК ООО "НПФ "Диагностические приборы"
и сертификации в СЦП ПНК по схеме EN 4179

Заявитель, желающий пройти аттестацию и сертификацию должен предоставить в ООО "НПФ" Диагностические приборы "следующие документы:

1. Заявку на сертификацию в соответствии с EN 4179:2017;

2. Копию диплома об образовании;

3. Сертификат обучения / квалификационная книжка, справка о программе (стандарт для программы / изученные предметы с указанием часов теории и практики) и имена преподавателей. Сертификат о соответствии учебного заведения;

4. Служебная записка от работодателя с отметкой места и продолжительности стажировки по каждому методу на основе 40-часовой рабочей недели (частные лица являются работодателем) по форме ФС.04

5. Справку о состоянии зрения по форме (ФС.05);

6. Код этики персонала неразрушающего контроля по форме (ДС.05);

7. Заявление о принятии правил по сертификации по форме (ФС.06);

8. Копию сертификата (если есть в наличии);

9. Особые требования по применению стандарта EN 4179 по форме (ФС.32);

10. Цветные фотографии 3х4 – 3 шт.

Комплект документов необходимо направить по адресу: 03061, г. Киев, ул. Патриотов, 103 (тел. 093-153-98-41)

Заказчик должен оплатить расходы на аттестацию и сертификацию специалиста по НК согласно договору.

После успешного прохождения сертификации специалисту выдается сертификат, квалификационное удостоверение и личная печать специалиста по НК.

Сертификат 1, 2 и 3 уровня выдается сроком на 5 лет.
